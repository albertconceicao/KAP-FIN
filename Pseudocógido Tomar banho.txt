

Objetivo: Tomar banho


Entrada 
Necessidade de tomar banho

Processamento
Pegar a toalha, transformar a necessidade em ação

Saída:

Passos para tomar banho:

-> Ter a necessidade de tomar banho
-> Pegar a toalha para tomar banho
-> Pegar roupas limpas para vestir
-> Verificar se tem os itens de higiene pessoal para tomar banho;
-> Tomar banho 
-> Colocar a roupa suja dentro do cesto de roupas sujas
-> Sair do banheiro

-------Pseudocódigo------------

Entrada:

Necessidade de tomar banho

Processamento:

-> Pegar a toalha
		-> Se a toalha estiver suja então coloque dentro do cesto de roupa suja, caso contrário use a toalha
		-> Pegue a toalha limpa;
-> Pegar roupas novas para utilizar após o banho:
						-> Se for sair depois o banho, então pegue roupas de sair;
								-> Se essas roupas estiverem amassadas, então passe elas;
						-> Se for dormir após o banho, então pegue roupas de dormir
						-> Se for ficar em casa, então pegue roupas de ficar em casa;
-> Verificar se tem todos os itens necessários para tomar banho:
						-> Se tiver todos os itens necessários, então tome o banho, caso contrário verifique na dispensa;
												-> Se não tiver na dispensa, então coloque na lista de compras;
						-> Ir para o banheiro;
-> Tomar banho
		-> Se tomar o banho, então seque com uma toalha;
		-> Aplicar desodorante e jogar a roupa suja no cesto;

Saída:

-> Banho tomado;
